Contact Page
This is a simple contact page designed to collect user information, including their name, email, address, and message. The page also includes contact information and social media links.

Features
Contact Form: Collects user information such as name, email, address, and message.
Contact Information: Displays phone number, email, and physical address.
Social Media Links: Provides links to social media profiles.
Responsive Design: The page is designed to look good on both desktop and mobile devices.
Preview
You can view a live demo of the contact page by navigating to Contact Page.

Technologies Used
HTML: Structure of the webpage.
CSS: Styling of the webpage.
Icons: Font Awesome for social media icons.
